수리시장용 건적 시스템 지원

전자부품 단가와 MC 자료를 iPhone에서 검색하고 로컬로 관리하는 업무용 견적 DB 앱입니다.

주요 기능

  • 제품명, ERP 코드, 순번 기반 단가 검색
  • MC 자료와 재료비율 확인
  • 항목 추가, 수정, 삭제 및 가격 이력 관리
  • JSON 백업 가져오기와 내보내기

자료 관리

앱의 자료는 기기 내부 저장소에 저장됩니다. 새 기기에서 사용하거나 백업이 필요한 경우 앱의 DB관리 화면에서 JSON 파일을 내보내거나 가져오십시오.

문제 해결

  • 검색 결과가 보이지 않으면 검색어를 줄이거나 ERP 코드 일부로 다시 검색하십시오.
  • 가져온 자료가 보이지 않으면 JSON 백업 파일 형식이 앱에서 내보낸 형식과 같은지 확인하십시오.
  • 앱을 삭제하면 기기 내부에 저장된 로컬 자료도 함께 삭제될 수 있으므로 삭제 전 백업을 권장합니다.

개인정보

이 앱은 계정 로그인, 외부 서버 동기화, 광고 추적을 사용하지 않습니다. 자세한 내용은 개인정보 처리방침을 확인하십시오.
